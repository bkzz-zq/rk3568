# 云服务器中转下载指南

> 国内访问 GitHub 慢，但云服务器（阿里云/腾讯云）下载 GitHub 很快。
> 利用云服务器做中转，快速将文件传到本地电脑或 RK3568。

## 原理

```
GitHub → 云服务器（快） → 本地电脑/RK3568（内网传输，快）
        ↑ 国内服务器直连          ↑ 局域网或国内网络
          海外节点                 不走国际线路
```

---

## 方法一：云服务器开临时 HTTP 下载（最通用）

适用：任何设备（RK3568、本地电脑、手机）都能下载。

### 第 1 步：云服务器下载文件

```bash
# SSH 登录云服务器
ssh root@<云服务器IP>

# 下载需要的文件（GitHub 在服务器上很快）
cd /tmp
wget https://github.com/fatedier/frp/releases/download/v0.61.1/frp_0.61.1_linux_arm64.tar.gz
wget https://github.com/fatedier/frp/releases/download/v0.61.1/frp_0.61.1_linux_amd64.tar.gz
# ... 其他文件同理
```

### 第 2 步：开临时 HTTP 服务

```bash
# 在文件所在目录开 HTTP 服务
cd /tmp
python3 -m http.server 8888
# 显示: Serving HTTP on 0.0.0.0 port 8888 ...
```

> ⚠️ 确保云服务器安全组已放行 8888 端口

### 第 3 步：从其他设备下载

**RK3568 上下载：**
```bash
wget http://<云服务器IP>:8888/frp_0.61.1_linux_arm64.tar.gz
```

**本地 Windows 电脑下载：**
```powershell
# PowerShell
curl -o frp_0.61.1_linux_arm64.tar.gz http://<云服务器IP>:8888/frp_0.61.1_linux_arm64.tar.gz

# 或浏览器直接打开
# http://<云服务器IP>:8888/
# 会列出所有文件，点击即可下载
```

### 第 4 步：关闭临时服务

```bash
# 下载完成后，Ctrl+C 关闭 python HTTP 服务
# 别忘了用完后关闭，避免安全风险
```

## 方法二：SCP 直传（云服务器 → 本地电脑）

适用：本地电脑直接从云服务器拉文件。

```powershell
# Windows PowerShell / CMD
scp root@<云服务器IP>:/tmp/frp_0.61.1_linux_arm64.tar.gz .

# 指定保存路径
scp root@<云服务器IP>:/tmp/frp_0.61.1_linux_arm64.tar.gz D:\Downloads\
```

然后再从本地传到 RK3568：

```powershell
scp D:\Downloads\frp_0.61.1_linux_arm64.tar.gz root@192.168.0.100:/tmp/
```

---

## 方法三：SCP 直传（云服务器 → RK3568）

适用：RK3568 能主动连云服务器。

```bash
# 在 RK3568 上执行
scp root@<云服务器IP>:/tmp/frp_0.61.1_linux_arm64.tar.gz /tmp/

# 或用 wget 从云服务器拉
wget http://<云服务器IP>:8888/frp_0.61.1_linux_arm64.tar.gz -O /tmp/frp_0.61.1_linux_arm64.tar.gz
```

## 常用下载场景

### FRP 内网穿透

```bash
# 云服务器上下载
cd /tmp
# AMD64（云服务器用）
wget https://github.com/fatedier/frp/releases/download/v0.61.1/frp_0.61.1_linux_amd64.tar.gz
# ARM64（RK3568 用）
wget https://github.com/fatedier/frp/releases/download/v0.61.1/frp_0.61.1_linux_arm64.tar.gz

# 开临时服务
python3 -m http.server 8888
```

### RKNN NPU 运行时

```bash
# 云服务器上下载
wget https://github.com/rockchip-linux/rknpu2/raw/master/runtime/Linux/librknn_api/aarch64/librknnrt.so

# RK3568 上获取
wget http://<云服务器IP>:8888/librknnrt.so -O /usr/lib/librknnrt.so
ldconfig
```

### Python 包（离线安装）

```bash
# 云服务器上下载 pip 包（ARM64 架构）
pip download -d /tmp/pkgs -d /tmp/pkgs \
    --platform aarch64 --python-version 310 \
    --only-binary=:all: \
    numpy opencv-python websockets

# 开临时服务
cd /tmp && python3 -m http.server 8888

# RK3568 上下载并安装
wget -r -np -nH --cut-dirs=1 -R index.html http://<云服务器IP>:8888/pkgs/
pip install --no-index --find-links=pkgs numpy opencv-python websockets
```

### 大模型文件

```bash
# 云服务器上下载
wget https://github.com/ultralytics/assets/releases/download/v8.1.0/yolov8n.pt -O /tmp/yolov8n.pt

# 开临时服务
cd /tmp && python3 -m http.server 8888

# RK3568 上下载
wget http://<云服务器IP>:8888/yolov8n.pt -O /opt/rk3568/models/yolov8n.pt
```

## 速查表

| 场景 | 云服务器命令 | 本地/RK3568 命令 |
|------|------------|----------------|
| 下载单个文件 | `wget <URL>` | `wget http://IP:8888/<文件名>` |
| 浏览所有文件 | `python3 -m http.server 8888` | 浏览器打开 `http://IP:8888/` |
| SCP 到本地 | - | `scp root@IP:/tmp/文件 .` |
| SCP 到 RK3568 | - | `scp 文件 root@192.168.0.100:/tmp/` |

## 注意事项

1. **安全组放行**：临时 HTTP 服务需要在云服务器安全组放行对应端口（如 8888）
2. **用完即关**：下载完成后立即关闭 HTTP 服务（Ctrl+C），避免暴露文件
3. **文件清理**：定期清理 `/tmp` 下的下载文件，避免磁盘满
4. **大文件注意带宽**：云服务器下行带宽通常 100Mbps+，但从云服务器到你本地的速度取决于云服务器上行带宽（3Mbps = 约 375KB/s）
