# 智能家居整合指南 — STM32 + App 改动

## 整体架构变化

```
改造前:
  App → OneNET API (heclouds.com) → MQTT → STM32
  App → 腾讯云 FastAPI (43.142.21.233:8000)  ← 认证+历史数据

改造后:
  App → RK3568 FastAPI (:8000) → 本地 MQTT → STM32
         ↑ 认证+设备状态+控制+历史数据+AI视觉（全部本地化）
```

## 一、STM32F407 修改（2 个文件）

### 1.1 `lwip_demo.c` — 改 broker 地址

```c
// ============ 改前 ============
#define PORT              1883
#define IP_ADDR           "183.230.40.96"
#define ONENET_MQTT_HOST  "mqtts.heclouds.com"

const char devPubTopic[] = "$sys/0g0k8iPR77/esp8266/thing/property/post";
const char *devSubTopic[] = {"$sys/0g0k8iPR77/esp8266/thing/property/set"};

// ============ 改后 ============
#define PORT              1883
#define IP_ADDR           "192.168.0.100"        // ← RK3568 IP
#define ONENET_MQTT_HOST  "192.168.0.100"        // ← RK3568 IP（或用 IP_ADDR 直连）

const char devPubTopic[] = "device/esp8266/property/post";      // ← 简化 topic
const char *devSubTopic[] = {"device/esp8266/property/set"};    // ← 简化 topic
```

### 1.2 `onenet.c` — 简化 MQTT 认证

标准 mosquitto 不需要 OneNET 的 Token 认证。修改 `OneNet_DevLink`：

```c
// ============ 改前 ============
#define PROID "0g0k8iPR77"
#define TOKEN "version=2018-10-31&res=products%2F0g0k8iPR77%2Fdevices%2Fesp8266&et=1793536477&method=md5&sign=b75f1j7mkrzGCynUdiy%2FPg%3D%3D"
#define DEVID "esp8266"

// ============ 改后 ============
#define PROID "esp8266"       // ← client_id（简化）
#define TOKEN ""              // ← 无密码（局域网内无需认证）
#define DEVID "esp8266"       // ← 保持不变
```

### 1.3 JSON 格式 — 不需要改！

STM32 发布的 JSON 格式保持不变，RK3568 MQTT 客户端已兼容：

```json
{"id":"123","params":{"hum":{"value":60},"tremp":{"value":25},"position":{"value":50},"limit_sw":{"value":0}}}
```

控制命令 JSON 也保持不变：
```json
{"params":{"led":true}}
{"params":{"led":false}}
{"params":{"stop":true}}
```

### 2.2 `pages/index/index.js` — 改设备数据查询

```javascript
// ============ 改前（OneNET API）============
config: {
    authorization: "version=2018-10-31&res=...",
    product_id: "0g0k8iPR77",
    device_name: "esp8266",
    getinfo_url: 'https://iot-api.heclouds.com/thingmodel/query-device-property?...',
    setinfo_url: 'https://iot-api.heclouds.com/thingmodel/set-device-property'
},

Onenet_GetInfo: function() {
    wx.request({
        url: this.config.getinfo_url,
        header: { 'authorization': this.config.authorization },
        ...
    });
},

// ============ 改后（RK3568 FastAPI）============
// 删除 config 中的 OneNET 配置

Onenet_GetInfo: function() {
    var api = require('../../utils/api');
    api.request({
        url: '/thingmodel/query-device-property?product_id=0g0k8iPR77&device_name=esp8266',
        method: 'GET'
    }).then(function(res) {
        // res.data 格式与 OneNET 完全兼容！
        // {"data": [{"identifier":"tremp","value":"25"}, ...]}
        var data = res.data;
        // ... 后续解析逻辑不需要改
    }).catch(function() {
        that.setData({ loading: false, errorMessage: '网络请求失败' });
    });
},
```

## 三、RK3568 部署

### 3.1 安装依赖

```bash
# SSH 到 RK3568
ssh root@192.168.0.100
cd /opt/rk3568
source .venv/bin/activate

# 安装 mosquitto MQTT broker
apt install mosquitto mosquitto-clients

# 安装 Python 依赖
pip install fastapi uvicorn paho-mqtt PyJWT

# 配置 mosquitto 允许局域网连接
# /etc/mosquitto/mosquitto.conf 添加:
# listener 1883
# allow_anonymous true
systemctl restart mosquitto
```

### 3.2 启动

```bash
# 正常启动（IoT 服务随 main.py 自动启动）
python3 src/main.py --no-ocr

# 验证 FastAPI
curl http://localhost:8000/
# → {"service":"RK3568 智能家居服务器","version":"1.0.0","docs":"/docs"}

# 验证 MQTT
mosquitto_sub -t "device/+/property/post" &
# 等 STM32 上线后应看到数据
```

### 3.3 API 文档

浏览器访问 `http://192.168.0.100:8000/docs` 查看完整 API 文档（Swagger UI）。

## 五、注意事项

1. **STM32 和 RK3568 必须在同一局域网**
2. **App 和 RK3568 必须在同一局域网**（手机连同一 WiFi）
3. 如需外网访问，可在路由器做端口转发
4. mosquitto 默认允许匿名连接，局域网内安全
5. 默认管理员账号: `admin` / `admin123`
